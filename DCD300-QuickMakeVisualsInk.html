Quick Make: Visuals & Ink

For this week's Quick Make, you will work with a tool called Inky -- a freely available platform for building hypertexts through mark-up practices (the primary platform behind Salt Immortal Sea) that outputs as HTML and CSS, making it an ideal platform for exploring both choice-based storytelling and visual content.


PROMPT: Using the Inky environment and Ink language, create a short story that includes 5 moments of choice for the reader to make. As well, create 3 images using whatever tools you have available -- drawing and scanning; drawing on a tablet / digital environment; photographs; etc. -- that somehow fit within your story. Following the Inky Tutorials and Guides, incorporate these images into your short story in a way that is visually and narratively interesting.* 
  *Note: Inky runs on web-based output logic. So you are basically creating a web page and setting up a web page folder directory. Look back to DCD 102, Google, and other DCD web development classes for a refresher on how to do this.
 

During this (and every) Quick-make, I encourage you to also explore writerly storytelling techniques -- think about what stories might be interesting here or how the images can fit in a way that is compelling; consider following the theme of this week and creating a short adaptation of a story you already know (fairy tales work really well for this!). As well, if you are familiar with Ink/Inky, I encourage you to try out a technique you have not used before -- Conditional content or custom CSS, for instance.


When you are done:

  1. Go to File > Export for web. This will create a website directory folder with (among other things) a .css, .js, and .html folder. (Your images & image folder will also need to be here in order for them to display in the Inky environment).
  2. ZIP / COMPRESS that ENTIRE FOLDER! Do not move anything or change anything. Just add your images, adjust your css (if you do any styling changes). 
  3. Submit the ENTIRE COMPRESSED FOLDER To Moodle.
